Birth Control Modeling for African Elephant Population

Skills: MATLAB, Differential Equation, Numerical Analysis

Objectives: Developed a mathematical model to stabilize an elephant population at 11,000 elephants using contraceptives in Africa.

Approach: Modeled the number of female elephants to dart with contraceptives by deploying a system of linear differential equations to find a steady-state solution in MATLAB.

Results: The number of female elephants to be darted varies by year since the contraceptives are only effective for 2 years. Please refer the report below for more details.
